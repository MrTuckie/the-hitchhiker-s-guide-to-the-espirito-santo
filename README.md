# the-hitchhiker-s-guide-to-the-espirito-santo

## o objetivo

criar uma espécie de rede social em python-django para que as pessoas registrem
informações sobre os lugares capixabas, assim como o guia do mochileiro das galáxias.

---

de humano para humanos.

## Conceito de Feito

### v.1.0

Para a primeira versão, para o cachorro quente ser um cachorro quente, eu vou precisar
que o sistema tenha:

- Um crud de lugares;
- Um crud de atividades dos lugares;
- Um crud de posts descritivos sobre os lugares.

Naturalmente, eu teria que colocar na mão as informações. Entretanto, é o mínimo cachorro quente possível.

> And why is that?

O guia do mochileiro das galáxias é basicamente uma wikipédia atualizada por, adivinhem só, mochileiros.
Ou seja, mochileiros vão aos lugares, adicionam verbetes aos lugares e eles viram a descrição do lugar.
Além disso, seria interessante para que os próximos mochileiros (apenas os insteressantes) que visitarão aquele lugar interessante
saibam o que vai estar acontecendo de interessante naquele luegar intereesante.

Ou seja, preciso ter o lugar, a atividade e as descrições dos usuários sobre o lugar. Penso em fazer para a atividade/banda/etc depois.
A próxima versão teria um sistema de votação que seria utilizado para elencar qual descrição seria a melhor, sendo que você poderia
ter acesso ao outros posts também.
